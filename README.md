# Datepicker - Lightweight and simple datepicker with no dependencies


## Options

```js
new Datepicker({
    selector: '.datepicker',
    lang: 'en',
    format: "mm/dd/yyyy",
    weekShort: true,
    multiplePick: true,
    disablePastDays: false,
    onLoad: () => {},
    onChange: () => {},
    onSelect: () => {}
});
```


## Browser support

- IE10
- Chrome 12
- Firefox 16
- Opera 15
- Safari 5.1
