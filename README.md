# Datepicker
### Lightweight and simple datepicker with no dependencies.

## Options

```js
new Datepicker({
    selector: '.datepicker',
    lang: 'en',
    format: false,
    weekShort: true,
    disablePastDays: false,
    multiplePick: true,
    minDate: false,
    maxDate: false,
    onLoad: () => { /** callback function */ },
    onChange: () => { /** callback function */ },
    onSelect: () => { /** callback function */ },
});
```

### Supported Browsers:

- IE10
- Chrome 12
- Firefox 16
- Opera 15
- Safari 5.1
